# ☕ Coffee Shop Android App (Kotlin + MVVM)

A modern coffee ordering Android application built using **Kotlin** and **MVVM architecture**.  
This app simulates a real-world coffee ordering system with cart management and dynamic pricing.

---

## ✨ Features

- 🚀 Splash Screen
- 📂 Category Listing
- 🔥 Popular Items Section
- 📄 Product Detail Page
- 🛒 Add to Cart
- 💰 Dynamic Cart Price Calculation

---

## 🛠 Tech Stack

- Kotlin
- MVVM Architecture
- RecyclerView
- ViewModel
- Repository Pattern
- JSON-based Local Data

---

## 🧠 Architecture Used

The app follows **MVVM (Model-View-ViewModel)** architecture:
- UI is separated from business logic
- ViewModel manages data
- Repository handles data source
